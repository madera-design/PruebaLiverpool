# Prueba Liverpool - Pokemon Browse

Aplicacion React en JavaScript para browse/search de Pokemon usando PokeAPI.

## Requisitos

- Node.js 22 o superior
- npm 11 o superior

Versiones usadas durante el desarrollo:

```bash
node v22.x
npm 11.x
```

## Instalacion

```bash
npm install
```

## Levantar En Local

Iniciar la aplicacion:

```bash
npm run dev
```

Abrir en el navegador:

```bash
http://127.0.0.1:5173
```

Backend simulado opcional con json-server:

```bash
npm run server
```

Disponible en:

```bash
http://localhost:3001/productExtras
```
